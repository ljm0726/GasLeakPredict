# GasLeakPredict


## **프로젝트 개요**

- 2015년 부터 2019년 까지 도시 가스 사고가 급증한 상황에서 직접적인 요인만을 고려함. 우리는 간접적인 영향을 미칠 수 있는 환경을 고려하여 가스 누출 사고와 환경이 미치는 상관관계를 찾아 예측하는 기능을 제공


## 프로젝트 기간

- 2020.10 ~ 2020.12
